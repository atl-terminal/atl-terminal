# Contributing

Thank you for testing ATL Terminal.

Useful feedback includes:

- Windows version and install type: portable ZIP or setup installer
- SSH/Telnet target type: Linux, Cisco IOS, jump host, lab device, etc.
- What worked
- What failed
- Screenshots or sanitized terminal output
- Steps to reproduce

Please remove secrets before posting logs or screenshots.

Plugin contributions belong in:

- https://github.com/atl-terminal/atl-terminal-plugins
- https://github.com/atl-terminal/atl-terminal-plugin-sdk
